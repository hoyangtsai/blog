## 404: Page not found

Sorry, we've misplaced that URL or it's pointing to something that doesn't exist. [Head back home]({{ site.baseurl }}/) to try finding it again.

<script>
  const deadPaths = ['/blog/2022/01/03/javascript-algorithm-quickselect/'];
  const newSite = '//hoyangtsai.github.io/traverse';
  const path = window.location.pathname;
  if (deadPaths.includes(path)) {
    window.location.replace(`${newSite}/${path.split('/').filter(Boolean).pop()}`);
  }
</script>
