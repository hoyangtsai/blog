小編搭乘的樂桃航空到達關西機場時<br>
停靠的是第二航廈（T2）<br>
在停機坪上下飛機後步行進航廈<br>
{% include post_image.html src="/images/osaka-trip/IMG_0085.JPG" alt="flypeach-airplane" width="100%" height="auto" %}

在飛機上空姐會發放入境卡跟申告單<br>
可以事先領取，在飛機上填寫好<br>
如果沒有領取到也沒關係，在辦理入境前會有地方可以領取、填寫<br>

不要傻傻的跟小編一樣<br>
覺得臺灣護照在日本免簽就可以像入境韓國一樣什麼單據都不需要填寫 😆<br>

關於填寫入境卡跟申告單有幾個需要注意的小細節：<br>
<ol>
  <li><b>事先準備好入住民宿/飯店資料</b><br>
    入境卡上需要提供入住民宿/飯店的名稱、電話，若不是一下飛機就有網路的狀態會很麻煩。建議可以事先準備好。
  </li>
  <li><b>申告單可多人共用</b><br>
    若是家族出遊可多人共用一張申告單，走同一個通道；情侶則需各自填寫。海關會在檢查沒問題時將申告單給收走
  </li>
</ol>
<br>
至於，入境卡與申告單的填寫小編這邊就不多做說明，現場都有很清楚的填寫範例～～

入境後接著要做的事情就是離開機場<br>
首先要去第一航廈（T1）<br>
兩個航廈之間有段距離<br>
機場有提供免費的接駁車往返T1、T2之間

在完成入境後<br>
一踏出門看到的是一間7-11<br>
抬頭仔細看會有標示告知接駁至T1的巴士乘坐地點<br>
{% include post_image.html src="/images/osaka-trip/shuttle-bus-sign.png" alt="shuttle-bus-sign" width="100%" height="auto" %}

附上一張關西機場的地圖，上頭有清楚標示著巴士的搭乘地點<br>
接駁巴士約5～8分鐘一班<br>
{% include post_image.html src="/images/osaka-trip/airport-map01.jpg" alt="airport-map" width="100%" height="auto" %}

到了T1後<br>
就可以依照個人的目的地選擇需要搭乘的交通工具

在日本比較常見的IC卡就是ICOCA和PiTaPa<br>
若是自由行的人建議可以先在關西機場自動販賣機購買一張ICOCA卡<br>
{% include post_image.html src="/images/osaka-trip/icoca.jpg" alt="icoca" width="100%" height="auto" %}

後續可以減少一些排隊買票的時間<br>
這張卡有點像是悠遊卡一樣<br>
搭乘公車、地鐵、火車都可以用

使用ICOCA有幾個細節需要注意一下：<br>
<ol>
  <li>押金500日圓<br>
    首次購買ICOCA時，選定的金額裡有500日圓是押金（可在退卡時退還）。意思就是說，如果在自動販賣機上點選購買額度2000日圓的ICOCA卡，扣除押金，實際能使用的只有1500日圓。
  </li>
  <li>無法賒帳<br>
    ICOCA卡不能像悠遊卡允許賒帳一次，事後儲值扣款。在餘額不足的狀態下是無法進出閘門的，必須先到周邊的IC卡精算機進行儲值方能使用。
  </li>
  <li>票價不便宜<br>
    大阪地鐵票價其實並不便宜，這六天五夜的大阪行小編記憶中我們一直在補票。如果有特殊的需求，其實有很多套票可供選擇，會比較划算。 ex.大阪地鐵一日票
  </li>
</ol>

接下來小編要分享一些其他的交通資訊啦～～～<br>
主要分成兩個：大阪 <->奈良 & 大阪 -> 關西機場

這六天五夜的大阪行<br>
我們依靠的是 Google map<br>
講到這個真的不得不誇獎 Google map 實在愈來愈強大了<br>
輸入了目的地後，會提供對應的大眾交通工具時刻表以及價格<br>
這些方便的資訊對於自由行的人真的是一大福音<br>
{% include post_image.html src="/images/osaka-trip/IMG_5989.png" alt="google-map-train-timetable" width="100%" height="auto" %}
### 大阪 <-> 奈良
從大阪到奈良可以選擇搭乘近鐵奈良線<br>
附圖是各種列車的停靠站<br>
{% include post_image.html src="/images/osaka-trip/namba-nara-stops.jpg" alt="namba-nara-stops" width="100%" height="auto" %}

這邊<u>千萬千萬千萬不要</u>選擇特急<br>
特急與快速急行的停靠站只差<b>兩站</b><br>
但是價格卻是差了好多<br>
選擇特急列車，除了原先的票價外，還要購買特急列車券（因為在進站時有刷了IC卡，特急列車的票在月台購買）<br>
{% include post_image.html src="/images/osaka-trip/IMG_5803.JPG" alt="IMG_5803" width="100%" height="auto" %}

另外，在買票的時候也要特別注意<br>
購買成人票記得要選擇左邊的選項<br>
千萬別像小編一樣選擇了右邊，買了孩童票，上車被驗票真的是很尷尬。<br>
{% include post_image.html src="/images/osaka-trip/IMG_5805.JPG" alt="IMG_5805" width="100%" height="auto" %}

### 大阪 -> 關西機場
小編這次的大阪行都是住在<a href="http://www.superhotel.co.jp/s_hotels/honmachi/" target="_blank">スーパーホテルLohas地下鉄四つ橋線・本町24号口</a><br>
{% include post_image.html src="/images/osaka-trip/IMG_5780.JPG" alt="スーパーホテルLohas" width="100%" height="auto" %}
飯店的門口緊鄰地鐵口非常方便！！
{% include post_image.html src="/images/osaka-trip/IMG_0554.JPG" alt="スーパーホテルLohas" width="100%" height="auto" %}

因為回程這天，早上一些行程的耽誤<br>
只好從飯店出發直接到關西機場<br>
出發前一樣先用 Google map 規劃好路線<br>
因為我們飯店的地鐵口下去離我們最近的是藍色四つ橋線<br>
比較一下時間和價錢，我們選擇了路線4<br>
{% include post_image.html src="/images/osaka-trip/IMG_0767.PNG" alt="IMG_0767" width="100%" height="auto" %}
{% include post_image.html src="/images/osaka-trip/IMG_0768.PNG" alt="IMG_0768" width="100%" height="auto" %}

但是沒想到在這吃了點虧...<br>
在岸里下車後原本以為可以從地下道直通天下茶屋<br>
結果是要從地鐵走上平面道路，才可以到換乘車站 😣<br>
(還好日本的路都很好走，拖著一堆行李也不會覺得累)<br>

上了南海特急列車後，只要30分鐘就可以直達關西空港<br>
出了閘門過個天橋就能到達第一航廈，非常方便～
{% include post_image.html src="/images/osaka-trip/IMG_0628.JPG" alt="南海特急列車" width="100%" height="auto" %}

如果行李很多、趕時間的人<br>
不仿考慮搭乘特急列車<br>

但像前面所說的，需要額外支付一筆費用（920日圓）<br>
