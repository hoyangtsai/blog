選擇國家/地區，選擇是否接受特別優惠的最新消息，同意《服務條款》

https://console.cloud.google.com/freetrial


區域，分別為
asia-northeast 在日本
asia-east 在台灣
asia-southeast 在新加坡

其他區域可以在這看

https://cloud.google.com/about/locations


機器類型，建議選擇微型就好，因為價差非常的大，有另外一個帳號，其中一個 instance 在使用幾天後，Google Cloud 平台建議將微型 vCPU 升級到小型，用了一個多月之後試用額度從 300 元扣到只剩 215 元左右
